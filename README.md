## Realtime USD to INR Currency Converter

Retrieves realtime value of Indian Rupee - ₹ (INR) for unit US Dollars (USD).

### Installation

```javascript
npm install realtime-usd-to-inr
```

### Usage

```sh
usd2inr
# $1 = ₹ 68.655

# and also you can provide the multiplication factor
usd2inr 5
$5 = ₹ 343.275
```
